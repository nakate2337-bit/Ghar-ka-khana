=== ASAN TIFFIN APP - INSTALLATION GUIDE ===

FILES IN THIS PACKAGE:
  index.html         → Main app
  manifest.json      → App install config
  sw.js              → Offline support
  icon-192.png       → App icon (Android)
  icon-512.png       → App icon (large)
  apple-touch-icon.png → App icon (iPhone)
  netlify.toml       → Hosting config

HOW TO INSTALL AS APP ON PHONE:

STEP 1 — HOST THE APP (FREE, takes 2 minutes):
  1. Go to https://netlify.com
  2. Sign up free with email/Google
  3. Click "Add new site" → "Deploy manually"
  4. Drag and drop this ENTIRE FOLDER onto the page
  5. Netlify gives you a link like: https://xyz-abc.netlify.app
  6. Open that link on your phone!

STEP 2A — INSTALL ON ANDROID (Chrome):
  1. Open the Netlify link in Chrome
  2. Tap the banner "Add to Home Screen" OR
     tap 3-dot menu → "Install App"
  3. Tap Install → Done!
  4. App icon 🍱 appears on home screen

STEP 2B — INSTALL ON iPHONE (Safari):
  1. Open the Netlify link in Safari
  2. Tap Share button (box with arrow ↑) at bottom
  3. Scroll and tap "Add to Home Screen"
  4. Tap "Add" → Done!
  5. App icon 🍱 appears on home screen
